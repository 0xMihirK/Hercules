Rating: C (Moderate Risk)

The environment displays significant indicators of technical debt. While no active breaches were identified, the presence of legacy, end-of-life software creates a clear path for exploitation.

  • Exploitable Legacy Infrastructure: The web server utilizes software from 2014, leaving the organization exposed to well-documented cyberattack techniques that could facilitate unauthorized system access.
